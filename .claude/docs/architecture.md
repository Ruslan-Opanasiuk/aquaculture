# Архітектура Aquaculture — пам'ять проєкту

Технічна вікі. Звичайний Markdown із `[[wiki-links]]` — можна відкрити папку
`.claude/` як Obsidian-сховище. Тримай це актуальним: тут «чому», а не лише «що».

## Огляд
SPA на React 19 + Vite. Преміальний магазин ікри, кураторський каталог. Дані —
статичні в `src/data/`, кошик — у Zustand, email/гурт — через `src/server.jsx` (Resend).
Хостинг — Vercel. Зображення — WebP 1x/2x через Sharp.

## Потік даних
```
data/catalogData ─▶ Catalog ─▶ CatalogSection ─▶ ProductCard
data/caviarPackages ─▶ Product (пакування, калькулятор, знижки)
components/discount ─▶ OrderSummary / Cart (логіка знижок)
store/cartStore (Zustand) ─▶ Cart, Header (лічильник)
data/seoConfig ─▶ components/SEO (helmet) ─▶ кожна сторінка
server.jsx (Resend) ─▶ wholesale endpoint  (поки не під'єднано з фронту)
```

## Сторінки
| Маршрут | Файл | Що |
|---|---|---|
| `/` | `pages/Home` | герой, фічі, відгуки, партнери |
| `/catalog` | `pages/Catalog` | 3 секції: червона / чорна / біла ікра |
| `/product/:id` | `pages/Product` | пакування, калькулятор, знижки |
| `/cart` | `pages/Cart` | підсумок, знижки, замовлення |
| `*` | `pages/NotFound` | 404 |

## Критичні блокери (не запускатись без них)
1. ~~**Checkout**~~ — ✅ `api/checkout.js` + форма в `Cart.jsx`: ім'я, телефон, email, місто,
   коментар → лист власнику зі складом кошика.
2. ~~**WholesaleForm**~~ — ✅ `api/wholesale.js` + localStorage 24h block.
3. **Контент продуктів** — у 9 позицій однаковий `longDescription` та `indicators`.
   Потрібен унікальний контент.
4. **Ціни каталогу** — заглушка «від 1234.00 грн» для всіх (`data/catalogData`).
5. **Домен/email** — `server.jsx` має `no-reply@yourdomain.com`; canonical-URL заглушки.
6. **Порожній кошик** — лінк веде на `/` замість `/catalog`.

  але без верифікованого домену Resend дозволяє слати тільки на зареєстровану пошту.
  Після підключення домену запрацює автоматично.
